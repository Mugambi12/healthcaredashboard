## Prerequisites

- Node.js installed
- npm installed

## Installation and Running

1. Navigate to the project directory:

   ```sh
   cd Coalition Technologies/healthcareDashboard
   ```

2. Install dependencies:

   ```sh
   npm install
   ```

3. Start the application:

   ```sh
   npm run dev
   ```
