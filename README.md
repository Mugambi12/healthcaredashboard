### Prerequisites

- Node.js installed
- npm

### Installation and Running

1. Navigate to the project directory:

```
cd Coalition Technologies/healthcareDashboard
```

2. Install dependencies:

```
npm install
```

3. Start the application:

```
npm run dev
```
